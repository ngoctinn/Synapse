description: Codebase Exploration & Discovery
---

1. **Tìm kiếm Từ khóa**:
   - Sử dụng `grep_search` để tìm các từ khóa liên quan đến task.
   - Xác định các file chứa định nghĩa (class, function) và cách sử dụng.

2. **Khám phá Cấu trúc**:
   - Sử dụng `list_dir` để hiểu cấu trúc thư mục xung quanh các file quan trọng.
   - Sử dụng `view_file_outline` để nắm bắt cấu trúc logic của file.

3. **Vẽ Bản đồ Phụ thuộc (Mental Map)**:
   - Xác định các module liên quan (Import/Export).
   - Hiểu luồng dữ liệu giữa các thành phần.

4. **Báo cáo**:
   - Liệt kê danh sách các file cần quan tâm.
   - Ghi chú về mối quan hệ giữa chúng.
