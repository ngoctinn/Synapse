---
description: External Tools Integration & Discovery
---

1. **Xác định Nhu cầu Công cụ**:
   - Task cần công cụ gì? (Database client, API tester, Library documentation...).

2. **Tìm kiếm Công cụ (MCP & Web)**:
   - Kiểm tra danh sách MCP servers hiện có (`list_resources`, `list_tools`).
   - Tìm kiếm trên web (`search_web`) để tìm thư viện hoặc công cụ phù hợp.

3. **Tích hợp/Sử dụng**:
   - Hướng dẫn cách cài đặt hoặc sử dụng công cụ.
   - Nếu là thư viện, kiểm tra độ tương thích với Tech Stack hiện tại.

4. **Tài liệu hóa**:
   - Ghi lại cách sử dụng công cụ mới vào tài liệu dự án (nếu cần dùng lâu dài).
